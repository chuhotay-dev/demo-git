*Chú thích - wd: working directory - sa: staging area - lp: local repository 1. git init : Tạo ra local repo (.git) 2. git status : kiểm tra xem những thay đổi của code (biết được file đang nằm ở vùng nào) 3. git add [ten_file] : chuyển 1 file từ vùng wd sang vùng sa 4. git add . : chuyển tất cả file (có sự thay đổi) từ vùng wd sang vùng sa 5. git rm --cached [ten_file] : đưa một file từ vùng sa quay ngược lại vùng wd
